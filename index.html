DRUPAL LADDER


The Drupal Ladder contains lessons and materials to help people learn about and contribute to Drupal. The site was created by the Boston Initiative to help Drupal user groups develop and share and develop materials for learn sprints and issue sprints.



TEST PATCHES



- PREREQUISITES

~ Drupal8 Sandbox Project Installed and running.

~ Git Installed

~ Basic Git Knowledge




- STEPS

1. Go to the Issue Page.






2. Download the patch and save it to the root directory of the Sandbox Project.







3. Go to the site and give a look to the issue.








4. Check the branch on which you are using the following command:
git branch




5. Using Command line move into the Drupal8 Sandbox Project root directory.
eg: $ cd path/to/drupal_8_sandbox_for_drupal_ladder




6. Apply the patch using the command:
git apply 1326088-rewrite-desription-for-url-alias.patch






7. To check whether the patch worked or not, check the status of the repository using the following command
$ git status






8. If it shows the core/modules/path/path.module file is modified than the patch has been applied successfully.



9. Now go back to the site and check whether the issue has been resolved.






10. Once done with testing, the changes can be undo which are applied by the patch by using the following command:
$ git reset --hard
